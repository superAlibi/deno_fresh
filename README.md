## Usage

跑起来

```shell
deno task start
```

## envariables

- location: 阿里巴巴的重定向地址
- DBID: 本地连接远程denokv数据库的数据id
- PGHOST postgresql数据库地址
- PGORT postgresql数据库端口
- PGUSER postgresql数据库用户名
- PGPASSWORD postgresql数据库用户密码
- PGDATABASE postgresql数据库名
- AUTHING_SECRET_KEY authing密钥
- AUTHING_APP_ID authing应用id
